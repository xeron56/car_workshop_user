# car_workshop_user

A new Flutter project.

## Getting Started

### Build and Run the App

1. **Clone the repository:**
    ```sh
    git clone car_workshop_user
    cd car_workshop_user
    ```

2. **Install dependencies:**
    ```sh
    flutter pub get
    ```

3. **Run the app:**
    ```sh
    flutter run
    ```
